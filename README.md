# Minecraft-builder-calculator-2.0
Convert blocks, item, stairs into another types! Since 2.0 it has a GUI

<img width=25% height=25% alt="image" src="https://github.com/user-attachments/assets/24dd171c-175a-4ce8-a63f-ff0a7772ad6d" />

## How to use?

Run `install.bat` file and then you will have a shortcut on your desktop

If your programm has no minecraft font, install font manually from `assets\mc_lang_font.ttf`

## Advanced config

<img width=25% height=25% alt="image" src="https://github.com/user-attachments/assets/4e4b1c1e-5686-47a4-aba8-dbe663ff0b1b" />

If you open advanced config, it will look like this:

<img width=25% height=25% alt="image" src="https://github.com/user-attachments/assets/85b4e210-7eb1-4a3e-84d1-1409e185b6a4" />

It will look something like this:
```
{
"type": [1; 2],
"type2": [3; 5],
...
}
```
It means, that if you craft "type" using 1 block you will get 2 of "type"<br>
You can add your types to the programm using this example, just don't forget about commas, and that there is no comma at the end
